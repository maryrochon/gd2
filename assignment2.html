Assignment 2: Sketches of the letter "S"

Sketch 1

This sketch has three ellipses. Two circles that match the background colour that overlay a third black oval like ellipse. It took a lot of trial and error to create this abstract looking ’s’.

Sketch 2

My second sketch is based on the in class demonstration. To try and change it up a bit I played around with the random function and the size of the ellipse tool. This is an interactive sketch.

Sketch 3

This was a bit of a happy accident. The background was based on the in class demonstration, but it went wrong and didn’t create a static background. The actual ’s’ is created by two while loops that we learned in class. Both have been rotated and stretched to make the a shape. This took Looooooooooots of trial and error to figure out the right translation and rotation.

Sketch 4

I took what I learnt from the previous sketch and made a more discernible ’s’. To do this, I had to adjust the x and y coordinates in the rect function, to get the arc I want, then use translate instead to move the while loop.

Sketch 5

I really liked the while loops, so again I was playing with the size of the arc on the page, the shape used and the size of the shape itself.

Sketch 6

This sketch, is based on a p5 reference that when you draw slowly makes what you draw look kind of 3D. Again like the 2nd sketch this is an interactive sketch.

Sketch 7

This is based on a talk I had with a fellow classmate. They were wondering how I did my first sketch and how they could do that for their letter. I showed how this could be done with the rect function for more linear letters, hence resulting in a linear ’s’.

Sketch 8

This sketch is based upon the in class demonstration. I just increased the size of the first ’s’ and added a thin red stroke to rotating ’s’ so that when the ’s’ is done rotating it will still stand out.

Sketch 9

This is also based on the in class demonstration. I added a second if statement and had them both stop when it looked like they were 3 quarters of the way complete. It took some trial and error to figure out how to get the blue if statement to go in the opposite direction.

Sketch 10

Here I was going through the p5 reference page and seeing how I could apply any of them to my letter. I used the curveVertex reference and played around with where to put points. It was not as easy as I thought it would be.

Sketch 11

This sketch forever reminds me of Shrek because of my font choice. I used the p5 text reference this time, and this is where I found out the z-axis is a thing that exists. I know prefer using, or adding it with the x or y axis.

Sketch 12

Behold a deconstructed ’s’ using the arc tool. I am unfamiliar with radians so this took a lot of trial and area to get arc that I am satisfied with. I opted to rotate some as I couldn’t figure out how to change the starting point.

Sketch 13

After pointing my toe in with WEBGL(3D) in my 11th sketch, I wanted to try and construct a ’s’ using boxes. I used the lesson from class about classes and made the width an interactive part of this sketch. The lower the x-coordinate the thinner boxes are, and higher the x-coordinate the heavier the boxes are.

Sketch 14

Uses the same class from the previous sketch I used the lesson from class and applied it to this sketch, with the switching colours and rotating ’s’. However I made it rotate across all axis.

Sketch 15

This sketch is inspired by the 13th sketch. This time the height changes, but it is controlled by a slider in the top left corner. I learned this from the p5 reference as well.

Sketch 16

Clearly I really this boxy ’s’. Here I experimented with the slider some more. However there is three and they all interact with each other. They respectful represent r, b, and g from the colour codes. Therefore moving them will change the letter’s colour.

Sketch 17

Following in the same direction of the boxes, I created the letter also with cylinders and played around with their sixes. It takes a lot of trial and error to get the cylinders to properly align. In addition to simply just displaying the letter, this sketch is interactive due to the addition of the obitControl function. So you can drag around the letter and scroll on it as well.

Sketch 18

Playing with the shape from the previous, I opted for it to rotate on its own. Here I added all three rotate functions. During these rotations is when I noticed I hadn’t been careful when putting the shape together and there was some small mistakes that I had to correct like cylinders not actually touching or going over the edge.

Sketch 19

Here I used the particle function that we learned in class to form an ’s’. This took a LOT of trial and error and restarting as the shape was originally too small. I learnt that I needed to individually name and call each fountain which was such a pain.
